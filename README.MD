## Objetivo

Esta atividade tem como objetivo garantir que o estudante consiga:

- Configurar o ambiente de desenvolvimento Dart
- Executar programas Dart via terminal
- Utilizar Git e GitHub para versionamento de código
- Documentar um projeto simples com README

---

## Estrutura do Projeto

```
hello-world-dart/
├── hello_world.dart
└── README.md
```

---

## Pré-requisitos

Ter o **Dart SDK** instalado na máquina.

- Download: https://dart.dev/get-dart
- Verificar instalação:

```bash
dart --version
```

---

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/hello-world-dart.git
cd hello-world-dart
```

2. Execute o programa:

```bash
dart hello_world.dart
```
